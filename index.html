--- layout: module-index --- A library of customizable user interface modules built for mobile-first and tested across a wide range of devices, while also providing a seamless desktop experience. All modules require either the [Zepto](http://zeptojs.com/) or [jQuery](http://jquery.com/) JavaScript libraries. All modules available on [GitHub](https://github.com/mobify/mobify-modules) and published under an [MIT license](https://github.com/mobify/mobify-modules/blob/master/LICENSE). If you find any issues with the modules, feel free to [file an issue](https://github.com/mobify/mobify-modules/issues), or submit a pull request!
